Takeaway

Main Takeaway

Across the five charts above, the same pattern repeats: metadata is the strongest single source, poster images help, and synopsis text contributes less than expected.

  • Best model — M + I + C reaches AP 0.6627
  • Main finding — adding text does not improve the strongest combination
  • Use case — the model is most useful as an early ranking tool before broadcast
