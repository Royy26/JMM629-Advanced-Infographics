# Can an Algorithm Predict Anime Success?

An infographic webpage based on the paper *Multi-Modal Supervised Learning for Improving Japanese Anime Audience Ratings Prediction*.

This project presents a simplified poster-style summary of the study using five core visualizations:

- production over time
- feature ablation results
- pairwise modality performance
- model-use simulation
- final takeaway panel

## How to View Locally

Open `index.html` in a browser.

Because this project is fully static, it does not require a build step or backend server.

## GitHub Pages

This project is designed to be published with GitHub Pages.
