THE RESEARCH

So… Can Data Do What Insiders Do?

Industry veterans use gut feel, franchise history, and studio reputation to predict a hit. We asked: can a machine learning model replicate that signal — using only pre-broadcast data?

We collected data on 1,403 TV anime from 2000 to 2025, across four distinct information types:

📊
M — Metadata
Release year, studio, genre tags, broadcast slot. The foundation — captures market cycles and studio track records.
🎨
I — Image
Key visual poster features via CLIP (400M image–text pairs). Captures art style, color palette, and character design.
📖
T — Text
Synopsis embeddings via JINA-CLIP-v2. The story description — how the show pitches itself before airing.
🎬
C/P — Cast/Production
Voice actor and studio reputation via Wikipedia2Vec. The talent and production pedigree behind the lens.

WHAT MATTERS MOST?

Which data ingredients matter most?

🍳
Think of it like a recipe. We tested 7 combinations of 4 ingredients — Metadata (M), Image (I), Text (T), Cast/Production (C/P) — and measured which mix predicts chart hits best. This technique is called feature ablation: remove one ingredient and see if the result gets worse. If it does, that ingredient was doing real work.

TAKEAWAY

What did the algorithm learn?

"The model sees what industry insiders feel — visual identity and production pedigree outperform the story summary."
01
Metadata is king. Release slot, studio, and genre tags give the model its strongest signal. Context beats content — what's around a show matters more than the show's own description.
02
The poster matters — the synopsis doesn't. Visual identity (Image) adds ~3 AP points on top of Metadata. Text actually hurts the best model — noisy Japanese synopsis embeddings introduce irrelevant variance.
03
Best model: AP 0.666 (M + I + C). A strong industry baseline for early-season ranking — most useful before broadcast, when only the studio, poster, and cast/production are known.
