## Where to download these images

### Method A — MyAnimeList (recommended for poster art)
1. Go to **myanimelist.net**
2. Search the show name
3. On the anime page, right-click the cover image → "Save image as..."
4. Rename to match the filename above (e.g. `demon-slayer.jpg`)
5. Ideal size: anything between 225×318 and 450×636 px (3:4 ratio)

### Method B — AniList
1. Go to **anilist.co**
2. Search the show → right-click cover → "Save image as..."

### Conan specifically
- **Detective Conan** on MAL: myanimelist.net/anime/235
- For a full-body PNG with transparent background, search:
  `"detective conan" "render" OR "png" site:zerochan.net`
  or use the official Shogakukan art from the Detective Conan website.

### Image tips
- JPG is fine for all posters (smaller file size)
- PNG preferred for Conan so the transparent background works with the hero overlay
- Keep originals at full resolution — the CSS handles cropping/scaling
- If a file is missing, the site automatically shows the emoji fallback (✔)
